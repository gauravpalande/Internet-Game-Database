# Internet-Game-Database
Object Oriented Analysis and Design project similar to IGN with Amazon API support

Role: C# .Net Web Services and Web API Developer

Description:
   The objective of the C# .Net web application is the collection of all games across the internet and create a centralized hub for users and critics review. 
   The application showcases the developers knowledge with the Object Oriented Analysis and Designing. The collection of data is through web crawlers. The database operations are handled using Entity Frameworks, Delegates and Lambda Functions. The application also helps in locating and purchasing new games through the help of Amazon's web API.
